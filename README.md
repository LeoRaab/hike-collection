# Hike-Collection

Hike-Collection ist eine App zur Verwaltung von Wanderungen. Diese können mit Freunden geteilt werden.

Die App ist für die Mobile Ansicht optimiert.

### Ionic-Native Plugins

- Camera

- Clipboard

### Firebase Features

- Build   
  
  - Authentication
  
  - Firestore
  
  - Hosting
  
  - Storage

- Engage
  
  - Remote Config
